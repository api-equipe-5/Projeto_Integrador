Aula de meio Ambiente

Nesse projeto as crianças são estimuladas a trabalharem em grupo. A elas são apresentados assuntos que abordam questões sobre o meio ambiente. São feitas atividades como:passeios em parques, plantio de uma pequena horta, e passeata ecológica.

Aula de Artes

No curso de artes, os alunos são estimulados e instruídos a criar algo que ajude a desenvolver seu lado artístico. São disponibilizados materiais adequados como, lápis, papeis, telas de pintura e massa de modelagem.
